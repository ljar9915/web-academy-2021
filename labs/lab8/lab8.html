introduction to CSS
by leah jarvis

CSS works by changing the appearance of a webpage & the different contents in it. to use CSS, you need to link a CSS file to the HTML document. then you can change the attributes of each part of your document & make it look however you'd like. the best part: the same CSS file can be used for multiple HTML documents, so you can give multiple webpages the same appeareance.

css helped me to improve my sites by allowing to make my websites look visually pleasing & more cohesive. using css, i was able to add the same background image, font style & color to all of my labs so that i don't have to change it individually for all of them. using css, i was also able to change the width & font size of my text for all my labs so that it'd easier to read.
